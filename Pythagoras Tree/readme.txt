Instructions to run: change to directory of files and type make. The Tree applications takes length and recursion depth as command line arguments.
For example: 
./Tree 100 7 
generates a tree with a base square of 100 by 100 px and a recursion depth of 7.

sfml is required to run the project. This was completed for UMass Lowell's Commputing 4 class around 2019
